# Content Map

This file maps the SmilePro reference content model into reusable sections for Smile Dental Center.

## Header

Reference fields:

- New patient notice.
- Address.
- Phone.
- Book Now CTA.
- Logo.
- Navigation links.

Smile Dental needs verified values before launch. Keep placeholders in content files until the user supplies official details.

## Hero

Reference:

- Headline: Exceptional Dental Care.
- Supporting line: financing-oriented message.
- CTAs: Book Now and Call Now.
- Trust signal: Google rating with stars.
- Large team photo.

Smile Dental adaptation:

- Use a similar short, confident headline.
- Use team/interior photography from `public/images/`.
- Add rating only if verified.

## Services Grid

- Preventative.

Smile Dental adaptation:
