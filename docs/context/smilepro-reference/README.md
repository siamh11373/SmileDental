# SmilePro Reference Context

Use this folder as the implementation reference for building a Smile Dental website that closely follows the structure, mood, and responsive behavior of `https://smileprodentist.com/`.

## How To Use This Folder

1. Start with `implementation-brief.md`.
2. Review `layout-analysis.md` for the homepage section order.
3. Review `responsive-notes.md` before coding mobile and iPad layouts.
4. Use `functionality-map.md` for navigation, CTAs, cards, carousel, map, and contact behavior.
5. Use `design-tokens.json` as the initial visual token guide.
6. Open screenshots in `screenshots/provided/` and `screenshots/live/` while implementing.

## Reference Intent

Create a very similar site layout and interaction model, adapted for Smile Dental Center using our own assets, business details, and copy.

Do not rely on the reference site's exact logo, photos, patient names, or proprietary wording in production. Match the layout, hierarchy, colors, typography feel, spacing, and responsive behavior.
